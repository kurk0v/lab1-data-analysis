# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта

Первичный анализ датасета **"Stock Price Dataset & Forecasting (FAANG+)"** в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** Финансовые показатели и технические индикаторы акций крупнейших технологических компаний (Apple, Amazon, Google, Meta, Microsoft, Nvidia). Данные предназначены для анализа рыночных трендов и решения задач прогнозирования временных рядов (прогноз цены закрытия следующего дня).

**Цель лабораторной:** Проведение аудита качества данных, формирование паспорта датасета, визуальное исследование распределений и подготовка плана по улучшению качества данных для дальнейшего моделирования.

## Структура проекта

* `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook, содержащий код загрузки данных, поиск дубликатов и пропусков, расчет статистик и визуализацию (EDA).
* `report/quality_report.md` — детальный текстовый отчёт с описанием выявленных проблем (асимметрия, выбросы, расчетные пустоты) и планом предобработки.
* `data/` — папка с исходным CSV-файлом данных.

## Основные этапы работы

1.  **Загрузка и описание данных:** Формирование перечня из 19 признаков (OHLCV, технические индикаторы RSI, MACD и др.).
2.  **Аудит качества:** Проверка на наличие пропущенных значений, полных дубликатов и корректность типов данных.
3.  **Статистический анализ:** Исследование распределений ключевых признаков (выявление правосторонней асимметрии цен).
4.  **Визуальный анализ (EDA):** Построение гистограмм, диаграмм размаха (boxplot) для поиска выбросов и диаграмм рассеяния для поиска зависимостей.
5.  **Отчетность:** Формирование выводов о пригодности данных для задач интеллектуального анализа.
